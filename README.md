#  Credit Card Analytics Dashboard (Power BI)

##  Project Overview

This project presents an interactive **Power BI dashboard solution** for analyzing credit card customers and transaction data. The goal is to derive meaningful insights into customer demographics, spending behavior, and transaction trends.

The project consists of two dashboards:

* **Credit Card Customer Report**
* **Credit Card Transaction Report**

---

##  Objectives

* Analyze customer demographics and segmentation
* Understand credit card usage patterns
* Track transaction trends and spending behavior
* Build interactive dashboards with filters

---

##  Datasets Used

###  Customer Data

Includes:

* Customer ID
* Age, Gender
* Income
* Education Level
* Marital Status
* Occupation
* Ownership details (Car, House)
* Customer Satisfaction Score

###  Transaction Data

Includes:

* Transaction Amount & Volume
* Spending Category
* Card Category
* Weekly Transaction Data
* Credit Limit & Utilization

---

##  Tools & Technologies

* Power BI
* Power Query (Data Cleaning & Transformation)
* DAX (Data Analysis Expressions)

---

##  Data Preparation

* Removed null and duplicate values
* Standardized categorical data
* Converted columns to appropriate data types
* Created relationships between datasets using CustomerID

---

##  DAX Calculations

* Age Group (18–25, 26–35, etc.)
* Income Group (Low, Medium, High)
* Total Revenue
* Total Transactions
* Average Transaction Value
* Current Week Revenue
* Previous Week Revenue

---

##  Dashboard 1: Customer Report

###  Features:

* Gender-based filtering
* Age Group segmentation
* Income Group analysis

###  Visualizations:

* Customer distribution by Age Group
* Gender distribution
* Income segmentation
* Education & occupation analysis
* KPI cards (Total Customers, Avg Income, Satisfaction Score)

---

##  Dashboard 2: Transaction Report

###  Features:

* Gender filter
* Weekly transaction filtering
* Spending category filter

###  Visualizations:

* Revenue trends over time
* Spending by category
* Card category performance
* Transaction volume analysis
* KPI cards (Revenue, Transactions, Weekly comparison)

---

##  Key Insights

* Customer distribution varies significantly across age and income groups
* Spending patterns differ across categories
* Weekly trends highlight fluctuations in revenue
* Certain card categories contribute more to overall revenue

---

##  Challenges Faced

* Handling missing and inconsistent data
* Creating accurate DAX measures
* Designing clean and interactive dashboards
* Maintaining relationships between datasets
